===========================
=== Jeu des différences === 
===========================
===== Yaaqoub Semlali ===== 
===========================
Jeu des différences or spot the difference. The goal of this game is to find a number of difference between two near-identical images (SVG Images).
those images generated randomly.
in the start of the game the player can set up the game in the way he like, 
Height and width of the generated images.
Figures number and Errors number.
also he can chose the type of the svg image (Circle, Ellipse, Line, Polygon, Rectangle, Mix).
and the game difficulty, if the player chose easy mode, for each error there is 5 sec to find it, and in the hard mode for each error there is 10 sec.
